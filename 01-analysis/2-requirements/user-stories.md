# User stories

> Этап 2. Пользовательские истории. Формат: «Как <роль>, я хочу <действие>, чтобы <ценность>».
> Каждой истории — критерии приёмки.
>
> **Скоуп: клиентское приложение и API для него** (роль «Клиент»). Функции инструктора и
> владельца обеспечиваются существующей инфраструктурой и здесь не описываются.

| ID | Как (роль) | Хочу (действие) | Чтобы (ценность) | Критерии приёмки | FR |
| :-- | :-- | :-- | :-- | :-- | :-- |
| US-1 | Клиент | зарегистрироваться по имени и номеру телефона без пароля | быстро начать пользоваться без барьеров | Указав имя и телефон, получаю доступ; повторный вход — по номеру телефона; сложный пароль не требуется. | FR-1, FR-2 |
| US-2 | Клиент | видеть список свободных слотов прогулок | выбрать подходящую прогулку | Список показывает слоты с датой/временем, маршрутом, инструктором, ценой, всего/свободно мест; слоты без мест помечены или скрыты. | FR-9 |
| US-3 | Клиент | фильтровать список слотов | быстро находить подходящий вариант | Могу фильтровать по дате/периоду, типу маршрута (новичковый/опытный), наличию свободных мест, инструктору; фильтры комбинируются; список обновляется по выбору. | FR-38 |
| US-4 | Клиент | открыть карточку слота со всеми параметрами | понять детали перед записью | Карточка показывает дату/время, маршрут и тип, инструктора, цену, всего/свободно мест, доступность прокатных досок. | FR-9a |
| US-5 | Клиент | записаться на выбранный слот | забронировать место на прогулке | После подтверждения создаётся запись; свободные места уменьшаются; запись появляется в «Моих бронированиях». | FR-10 |
| US-6 | Клиент | при записи указать свою или прокатную доску | приехать со своим инвентарём или взять напрокат | Можно выбрать «своя» / «прокатная» для каждого места; «прокатная» уменьшает прокатный фонд, «своя» — нет. | FR-11, FR-14 |
| US-7 | Клиент | одной записью забронировать несколько мест (себя + гости) | записать компанию за раз | Можно указать 1–3 места в одной записи при наличии свободных; для каждого гостя выбирается вариант доски. | FR-12 |
| US-8 | Клиент | быть защищённым от записи сверх лимита | не приехать на занятое/несуществующее место | Нельзя забронировать больше, чем `min(свободные доски, потолок маршрута)` мест и больше свободных прокатных досок; при гонке запросов двойная бронь/овербукинг исключены. | FR-13, FR-15 |
| US-9 | Клиент | видеть список своих бронирований | контролировать предстоящие и прошедшие прогулки | Вижу свои записи (предстоящие и прошедшие) со статусом, параметрами слота, числом мест и вариантом доски. | FR-35a |
| US-10 | Клиент | отменить свою запись до старта | освободить место, если планы изменились | Запись можно отменить до старта; при ранней отмене (≥2 ч) места возвращаются в слот; при поздней (<2 ч) запись помечается «поздняя отмена», место не освобождается, штрафов нет. | FR-16, FR-17, FR-18 |
| US-11 | Клиент | видеть цену прогулки | понимать стоимость заранее | В карточке и при подтверждении показана цена; оплата офлайн (наличные/перевод), запись фиксируется без онлайн-оплаты. | FR-30 |
| US-12 | Клиент | получать напоминание о предстоящей записи | не забыть и не пропустить прогулку | Заблаговременно до старта приходит напоминание (push/SMS — канал уточняется; доставку обеспечивает существующая инфраструктура). *(Should)* | FR-33 |

> **Вне скоупа.** Истории инструктора и владельца, а также оценки инструкторов обеспечиваются
> существующей инфраструктурой и в клиентском приложении не реализуются.
