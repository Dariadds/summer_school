# Промт для багов
У меня есть проект на Kotlin Multiplatform (KMP). Нужно найти 3 разных бага в коде и описать их по шаблону из задания.

Проект находится в папке: summer-school-2026-feature-client\summer-school-2026

Структура проекта:
- androidApp/ - Android-приложение
- iosApp/ - iOS-приложение
- shared/ - общий код (логика, API, модели)
- webApp/ - веб-версия

Задание:
1. Найди 3 разных бага в коде. Баги могут быть:
   - Визуальные (цвета, размеры, отступы не соответствуют дизайну в Figma)
   - Логические (ошибки в валидации, неправильная работа API)
   - Потенциальные (TODO, FIXME, пустые блоки catch)
   - Типобезопасность (неправильные типы, null safety)
   - Производительность (лишние перерисовки, утечки памяти)

2. Для каждого бага создай отдельный .md файл в папке проекта с именем bug-1.md, bug-2.md, bug-3.md.

3. В каждом файле опиши по шаблону из задания:
   - Симптом
   - Требования
   - Исправление
   - Промпты
   - Проверка

Где искать в первую очередь:
1. Папка shared/src/commonMain/kotlin/ — логика приложения
2. Папка androidApp/src/main/res/ — цвета, строки, размеры для Android
3. Папка webApp/src/ — визуальные компоненты веб-версии
4. Поищи TODO, FIXME, // TODO, // FIXME во всех файлах
# Промт для фич
У меня Kotlin Multiplatform проект "Волна". Нужно реализовать 3 фичи.

## Фича 1: Оценка/рейтинг инструктора (P2)
После завершения прогулки пользователь может оценить инструктора:
- Шкала 1-5 (звёзды)
- Необязательный текстовый комментарий
- Оценка сохраняется локально (Settings)
- Оценка не показывается другим пользователям

Где менять:
- BookingDetailsStore.kt — добавить SubmitRating intent
- BookingDetailsScreen.kt — добавить кнопку и Bottom Sheet

## Фича 2: Недавние маршруты (P2)
При просмотре маршрута сохранять его в историю:
- Хранить до 5 маршрутов
- На главном экране показывать до 3 последних
- Без дубликатов (если маршрут уже есть — переместить в начало)

Где менять:
- SlotDetailsStore.kt — сохранять при просмотре
- SlotListScreen.kt — показывать секцию "Недавние"

## Фича 3: Избранные маршруты (P3)
Пользователь может добавлять маршруты в избранное:
- Иконка сердца на карточке и в деталях
- Отдельный экран в профиле "Избранное"
- Хранить локально (Settings)

Где менять:
- SlotListScreen.kt — добавить иконку сердца
- SlotDetailsScreen.kt — добавить иконку сердца
- ProfileScreen.kt — добавить пункт "Избранное"
- Создать FavoritesScreen.kt — новый экран

Для каждой фичи создай отдельный .md файл: feature-1-rating.md, feature-2-recent.md, feature-3-favorites.md.
