# Test Cases — Feature 3: Избранные маршруты

## Замечания
- В текущей реализации избранное хранится локально и доступно через экран профиля.
- Удаление из избранного выполняется без отдельного подтверждения.

### TC-FAV-001

**Feature:** feature-3-favorites

**Название**
Успешное добавление маршрута в избранное и переход к нему из списка

**Цель**
Проверить, что пользователь может добавить маршрут в избранное, увидеть его в списке избранного и открыть детали маршрута по тапу.

**Предусловия**
- Пользователь авторизован.
- На устройстве нет сохранённых избранных маршрутов.
- В приложении доступен список маршрутов.

**Шаги**
1. Открыть экран деталей маршрута.
2. Нажать иконку сердца для добавления маршрута в избранное.
3. Перейти в профиль.
4. Открыть раздел "Избранное".
5. Нажать на добавленный маршрут.

**Ожидаемый результат**
- Маршрут добавляется в список избранного.
- В профиле появляется раздел "Избранное" с сохранённым маршрутом.
- При нажатии маршрут открывается на экране деталей.

**Приоритет**
High

**Тип**
Functional

### TC-FAV-002

**Feature:** feature-3-favorites

**Название**
Повторное добавление того же маршрута не создаёт дубликаты

**Цель**
Проверить, что один и тот же маршрут нельзя добавить в избранное несколько раз.

**Предусловия**
- Пользователь уже добавил маршрут в избранное.
- Открыт экран деталей этого маршрута.

**Шаги**
1. Повторно нажать иконку сердца на экране деталей маршрута.
2. Перейти в раздел "Избранное".
3. Проверить список избранных маршрутов.

**Ожидаемый результат**
- В списке избранного не появляется дубликат маршрута.
- Состояние избранного остаётся корректным.
- Маршрут отображается в списке один раз.

**Приоритет**
Medium

**Тип**
Negative

### TC-FAV-003

**Feature:** feature-3-favorites

**Название**
Пустое состояние списка избранного

**Цель**
Проверить, что при отсутствии сохранённых маршрутов пользователь видит корректное пустое состояние.

**Предусловия**
- Пользователь авторизован.
- В локальном хранилище нет избранных маршрутов.

**Шаги**
1. Перейти в профиль.
2. Открыть раздел "Избранное".

**Ожидаемый результат**
- Отображается состояние "Пока нет избранных маршрутов" или аналогичный пустой экран.
- Список пустой, без ошибок и без лишних элементов интерфейса.

**Приоритет**
Medium

**Тип**
UI
