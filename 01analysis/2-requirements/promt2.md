# Требования
На основе брифа и Q&A сгенерируй 5 файлов:

1. business-requirements.md (BR, приоритеты P1-P4, источники)
2. functional-requirements.md (FT по группам: расписание, авторизация, бронирование, отмена, оплата, оценка, уведомления)
3. non-functional-requirements.md (платформы, push, сроки, API)
4. use-cases.md (актёр, поток, альтернативы)
5. user-stories.md (роль, действие, ценность, FT)
